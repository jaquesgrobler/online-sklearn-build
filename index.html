NiLearn is a software package to facilitate the use of statistical learning on NeuroImaging data.

It leverages the scikit-learn Python toolbox for multivariate statistics with applications such as predictive modelling, classification, decoding, or connectivity analysis.

It requires only nibabel and scikit-learn.

Warning

NiLearn is still an unreleased package in early development stages.
